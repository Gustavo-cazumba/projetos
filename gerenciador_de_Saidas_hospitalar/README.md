# gerenciador de saida
"Sistema inteligente para gerenciamento centralizado e otimização do fluxo de saídas hospitalares, garantindo mais eficiência e comunicação entre equipes."

# funcionalidade
- Gerenciamento de Escalas: Controle e visualização das escalas de motoristas e      técnicos  de enfermagem.
- Solicitações em Tempo Real: Acompanhamento e gerenciamento das saídas inter-hospitalares.
- Notificações Automáticas: Alertas para a equipe sobre novas saídas e horários.

# tecnologia utilizada
- C#
- MYSQL
